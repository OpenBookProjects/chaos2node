Joyent
==============

粗看一下 `Joyent`_ 文档
就可以知道,这是家非常 NB 的云服务公司!支持很多非常前沿的技术,包括 `node.js`

- 的确! `Node.js开源项目获得Joyent资助，发展进入快车道 <http://www.infoq.com/cn/news/2010/11/nodejs-joyent>`_
- 而结合后的产物就是: `no.de`_  ~ 有爱的最短 `node.js` 应用托管服务!



.. sidebar:: BTW:
    :subtitle: 白菜价的商用维基

    - 注意到 `no.de`_ 的文档使用了 `Confluence <http://www.atlassian.com/software/confluence/overview>`_ 维基
    - Atlassian 这公司以 JAVA 的BBS 产品起家
    - 改造成 JIRA 后,以极其便宜的价格占领了市场后
    - 现在的产品覆盖了软件过程中的所有阶段,品质也很有保障,越來越多的企业选择了他们


参考文档:

- `Getting Started with a Node.js Development SmartMachine - Node - Joyent Customer Wiki <http://wiki.joyent.com/display/node/Getting+Started+with+a+Node.js+Development+SmartMachine>`_
- `npm Integration - Node - Joyent Customer Wiki <http://wiki.joyent.com/display/node/npm+Integration>`_
- 绝对,就可以快速使用起来!


**不过前提是:**

+ 有 UNIX/Linux 系统操作体验
+ 知道怎么生成 `SSH`_ 密匙对
+ 知道 `git`_ 的基础操作

好在,都不算是特别变态的前提,就不细说了,默认读者都具备了 ;-)
